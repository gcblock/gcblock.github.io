# 🔖 ios 配置openVPN客户端

# 前言

在阿里云轻应用服务器上安装了openVPN serve，在 window 10 系统上已经成功安装openVPN客户端并连接成功。

# 安装

在App Store上下载安装  openVPN ，方法自行百度。

logo 长这样噢>_<
