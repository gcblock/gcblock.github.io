# 🍉 圣火令

圣火令的首要目标是使用区块链技术实现引导资本发挥价值、财富合理分配，让天下资本专注民生和科学。火种是人类社会出现的一大工具，在今天圣火已经承载了人类美好意愿。今天我们通过区块链工具为人类铺垫道路，消灭二八定律，实现财富合理分配，故名圣火令。

# 🔖 引言

资本的虹吸作用、隧道效应让财富分配不均，竞争环境不公平，由此出现的二八定律不仅消灭了打工人的激情，更是重伤了一批一批敢于创业、拥有梦想的年轻人。那如何解决这个问题？国家适合宏观调控、提供公平自由的市场环境、但并不适合直接参与市场经营，这样会搓失创业者的积极性、打击市场的激情，所以国家只能在宏观层面发挥作用。能细致解决这个问题的答案是使用区块链技术构造公开透明无腐败的环境，使用圣火令引导资本，给上层阶级的资本发挥价值，给底层解决带来生产激情，逐步消灭阶级差距，使全社会生产力增加数倍。

圣火令是民众的共识、从民生利益出发的解决办法，由此只要符合条件任何个人或组织可以继承圣火令，众联商安投资管理有限公司无利润提供技术支持。

# 🔖 数字化运营单位

区块链技术可以解决作弊、腐败的问题，所以需要数字化运营单位提供一个公平、无监察投入的运营环境。

# 数字化办法概述

公司股权百分百量化为黄金托管合约，合约中标注的资产为黄金，基础单位为g，即一股等于 1 g 黄金，股东自由可自由通过区块链系统选择入退股本。 股东权利百分百基于区块链系统托管，量化的数字资产包含股本、权力、公共福利、即持有黄金托管合约即是单位股东。

# 股本

股本是投资者入场的条件，基本投资单位是黄金 g,选择黄金不仅仅是因为它具有上千年的流通历史和广泛的接受程度，更是最大程度保证投资者的财产不受场外环境的干扰、使投资者在任何环境任何时间可以全身进、全身退。

# 股权

股东权利分为分红和投票权两部分。为支持公司融资发展，除了特权分红部分，其余权力各个股东等同。

# ⭐️特权分红

为单位融资发展、激励阶级上层涌入，公司规定特权分红办法，特权分红最长时间不超过9年，超过时长不得续期，转为均股分红，准入条件由单位领导层决定。

# ⭐️均股分红

减去以特权股占比的分红，剩余分红在股东中均分。

# ⭐️ 均股投票

每个股东不论持有多少黄金托管合约，其投票权力都是一样的，即是均权投票。投票结果通过区块链产生，区块链是使用区块共识算法研发的区块链，既是黄金托管合约存储的区块链。

# ⭐️ 天使投资使用特权分红办法

为了更好的融资发展，当财团以融资的方式入场拥有特权分红权力，特权股享受按所持合约数量占比分红。

# ⭐️ VIP待遇

公司股东在享受公司资源时拥有VIP待遇。 VIP规则的制定由均权股东通过区块共识链投票决定。在生产力溢出或发生自然灾害，VIP待遇可以使股东在分配资源上达成共识，最终实现为股东提供福利或合理分配资源。另一方面，在公司正常的发展规划中，VIP待遇可以对公司的发展起到激励作用。

# 🔖 专利政策

开放的专利政策可以真正的给发明创造者提供福利，而不会让某项专利成为资本家暴利的砝码。

通过本公司产生的专利不存在垄断性，拥有统一计算规则:专利贡献者享受年总利润的百分之二十作为经济收入，享受年限根据法定专利机构评判。除此之外没有其他权利。

# 🔖 总经理权利

总经理是本公司最高职位，具有人事任免，投资运营等权力。

总经理经由一年一度的股东大会决定任免。当合约持有人认为总经理的行为对公司有害时可以通过区块链投票限制或罢免总经理的权利。

当总经理运营活动调用资金超过公司总资产的百分之30十需要提起区块链3天投票，当反对票超过百分之1时，投票时间延长12天。最终结果以支持票是否超过反对票决定此项事宜是否通过。

# 等价担保

公司运营有失败的风险，当公司运营失败、伤及股本时必须由有资质的担保方买单。即运营单位有多少股本，必须要多少担保资产在银行质押。

# 区块链认证

圣火令是基于区块链技术打造无监察投入的运营环境、公司涉及股东利益的项目必须进行数字化技术认证，确保使用的技术真实无漏洞，特别需要认证的模块有，股东投票模块、福利发放模块、利润计量公开模块、分红模块、税务模块、对外投资模块。

目前提供认证单位有众联商安投资管理有限公司、众联共识网络科技有限公司、后期经由GC链投票增加的认证单位。
