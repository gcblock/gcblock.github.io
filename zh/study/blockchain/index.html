# 🍉 区块链

# 引言

区块链是一个信息技术 (opens new window)领域的术语。从本质上讲，它是一个共享数据库，存储于其中的数据或信息，具有“不可伪造”“全程留痕”“可以追溯”“公开透明”“集体维护”等特征。基于这些特征，区块链技术 (opens new window)奠定了坚实的“信任”基础，创造了可靠的“合作”机制，具有广阔的运用前景。

2019年1月10日，国家互联网信息办公室 (opens new window)发布《区块链信息服务管理规定 (opens new window)》 [1] 。2019年10月24日，在中央政治局第十八次集体学习时，习近平总书记强调，“把区块链作为核心技术自主创新的重要突破口”“加快推动区块链技术和产业创新发展”。“区块链”已走进大众视野，成为社会的关注焦点。

2019年12月2日，该词入选《咬文嚼字》2019年十大流行语。 ---- 摘自百度百科 (opens new window)

# 区块链和共识算法

共识算法可以被看到实现区块链技术的核心算法，区块链的核心属性不可伪造、全程留痕、可以追溯（查询）、公开透明、集体维护是由“共识”完成的。如何完成呢，大家伙（分布式终端）在一起商议达成一个共同执行的章程（这是共识的由来），这个章程（算法）首先规定数据的存储公开透明，任何人都可以查询，有人试图伪造数据大家伙不认可，并禁止，甚至可以设定惩罚机制。具体如何实现众仙家各有法门，比如POW，POS……，各有利弊。

# ❤️ 比特币

比特币应用的是POW共识算法，故名思意是通过耗费能量来维持区块链的核心属性，再一个就是认为不可能有人（终端）掌握百分之51的算力来打破规则进行作弊。在这个充满二八定律的世界，我就呵呵了，充满了浓厚的资本主义臭味。

但比特币作为区块链技术的先驱者，其历史意义是不能被否定的，这也是众多投资者疯狂的原因吧。截至到现在比特币已经接近50，000美元大关。从发行到现在涨幅千万倍。一个特写的大字，服！
